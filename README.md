# crypto-alpha-research-system

## 一句话说明
一个带数据、回测、边界声明与验证协议的 crypto alpha 研究与纪律框架。

## 仓库定位
这个仓库发布的是一套 **crypto alpha research system** 的公开版材料，包含：
- 8 年历史 top1000 快照（统一 3 月 12 日口径）
- 跨周期规律提炼与交叉比对
- 评分卡、执行门、执行卡
- 系统边界声明与纸面组合验证协议

**重要边界：**
> 这不是一个被真钱验证过的投资系统。它是一个经过双周期内部检验、擅长排错、能约束执行冲动的 alpha 研究与纪律框架。

## 目录结构
- `references/`：框架、规律、回测报告、执行规则、边界与验证协议
- `data/`：8 年历史快照、候选池与回测表

## 仓库说明
- 数据为过滤后的可见市场快照，默认排除稳定币、wrapped 资产、staked 衍生币。
- 本仓库更适合用来研究 **alpha 仓位** 的排错与筛选，不适合作为 BTC/ETH 核心仓配置指南。
