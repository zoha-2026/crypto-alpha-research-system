# 跨周期规律交叉比对 v1
## 2019→2022 独立规律清单 vs 现有 2023→2026 规则系统

对照来源：
- 2019→2022 独立规律：`references/step2_3_rules_2019_2022_v1.md`
- 当前规则系统：`references/rules-change-table-v1.md`、`references/step3-rules-patch-v1.5.md`、`references/rules-scorecard-design-v2.md`

## 一、结论先说

这轮交叉比对后，我会把规律分成 3 类：

### A. 跨周期共性（最硬）
这些在两个周期里都独立出现，最值得升级成**结构性规律**：
1. **高位 incumbents 很难继续显著上升**
2. **赛道 / 生态命运先于个体胜负**
3. **真正可迁移的赢家，往往来自基础设施 / 入口 / 核心位置，而不是纯热度**
4. **创新浪潮会先给大量项目一起抬轿，随后快速分层淘汰**

### B. 周期特异性（只在某一轮特别强）
这些不能直接升级成普遍规律，更像周期环境的特征：
1. 2019→2022 的**新公链爆发**
2. 2019→2022 的**平台/分发资源红利**
3. 2019→2022 的**消费者注意力浪潮**
4. 2023→2026 的**RWA / 黄金锚定 / 真实收益资产偏好**
5. 2023→2026 的**供给与资本结构约束重新变得更重要**

### C. 2019→2022 独有的新发现（现有 17 条没完整覆盖）
最值得补进系统的，不是再多一条“会涨规律”，而是一个新的**Hard fail / 风险模式**：

> **反身性闭环飞轮 Hard fail**：
> 当代币需求过度依赖协议自身的收益飞轮、平台资产负债表、内部流动性支撑或自我增强叙事时，它可以在上升阶段成为最强加速器，但也可能在飞轮断裂时成为最脆弱的崩塌点。

LUNA 与 FTT 的完整轨迹，把这个盲区照出来了。

## 二、逐条对照：2019→2022 独立规律 vs 现有系统

| 2019→2022 独立规律 | 与现有系统的关系 | 判断 |
|---|---|---|
| 新公链 / 新生态爆发是周期主线 | 对应 `规律 4A` + `新规律 C（赛道有集体命运）` | **跨周期共性**，可升级为结构性规律 |
| 新物种（路径 B）是主战场 | 与 `新规律 A/B` 有关联，但 2019→2022 的强度更高 | **周期特异性**，但能强化“不要只盯旧龙头” |
| 平台 / 分发资源是独立 alpha 源 | 对应 `规律 2`，但 2019→2022 明显更强 | **周期特异性加强项** |
| 消费者注意力能制造 top50 | 与 `规律 6` 部分相关，但现有系统更偏警惕，不把它当核心模式 | **周期特异性** |
| DeFi 里入口型 / 基础设施型胜于一切都算 DeFi | 对应 `规律 1`，并提示其边界应更强调“入口与位置” | **跨周期共性** |
| 高位 incumbents 很难再进优先池 | 对应 `新规律 B（哑铃分布）` + `高位续涨折扣` | **跨周期共性** |
| 供给在 mania 里不是第一筛选器 | 对 `规律 7` 形成重要修正：供给是周期敏感变量，不是所有周期里都同样早起作用 | **新发现 / 需修正旧系统** |
| 反身性闭环飞轮可以制造最陡峭 ascent | 现有 17 条与评分卡里都没有完整覆盖 | **2019→2022 独有新规律 / Hard fail 盲区** |

## 三、跨周期共性：哪些可以升级为“结构性规律”？

### 1) 高位惯性 / 高位续涨难
证据：
- 2019→2022：高位爬升优先池 = `0`
- 2023→2026：高位强势项目在回测里反复出现 `FP`（TAO / PENDLE / KAS 等）

结论：
> **不管哪个周期，已经在 top50 左右的资产，继续显著前移都很难。**

这条已经不只是“提醒”，而是**结构性规律**。

---

### 2) 赛道 / 生态命运先于个体选择
证据：
- 2019→2022：新公链/生态整体爆发主导了最大一批赢家
- 2023→2026：`新规律 C` 明确指出赛道有集体命运，个体很难长期逆赛道

结论：
> **先选赛道，再挑项目** 不是口号，而是两个周期都独立跑出来的共同结果。

---

### 3) 入口 / 基础设施比“沾边叙事”更容易形成可迁移赢家模板
证据：
- 2019→2022：UNI / AAVE / GNO / RUNE 这类有明确位置的协议更能留下来
- 2023→2026：PENDLE / HYPE / JUP / MORPHO 等说明“有产品、有位置”依然是核心正例

结论：
> **真正能跨周期复用的，不是“这个赛道热”，而是“这个项目站在赛道关键位置上”。**

---

### 4) 创新浪潮会先一起抬，再迅速分层淘汰
证据：
- 2019→2022：DeFi Summer、L1 Summer、游戏/metaverse 都出现“先一起涨、后快速分层”
- 2023→2026：`规律 4B` 已经描述了技术叙事型资产 12-18 个月后的回落段

结论：
> **这其实是一个更广义的母规律：创新先行资产不会永远一起赢，浪潮过后会迅速分层。**

这意味着现有 `4B` 可以理解为这个母规律在“技术叙事型资产”上的一个特例。

---

## 四、周期特异性：哪些只属于某一轮？

### 1) 2019→2022：新公链爆发是主轴
- SOL / AVAX / LUNA / DOT / MATIC / NEAR
- 这是典型的基础层重新洗牌

### 2) 2019→2022：平台 / 分发资源格外重要
- LEO / FTT / CHSB
- 在创新牛市里，用户入口与交易分发权的价值被极大放大

### 3) 2019→2022：消费者注意力波极强
- MANA / SAND / AXS / SHIB / FLOW / BTT
- 这在 2023→2026 没有以同等强度重现

### 4) 2023→2026：RWA / 黄金锚定 / 真实收益偏好显著增强
- ONDO / BUIDL / XAUT / PAXG
- 这不是 2019→2022 的中心叙事

### 5) 2023→2026：供给与资本结构重新变得更“提前”
- WLD / ENA / STRK / TAO / JTO / KMNO
- 这轮市场对“低流通、高融资、资本是否在替代产品”更敏感

结论：
> **2019→2022 更像创新牛市，2023→2026 更像选择性更强、供给更受审视的成熟筛选市。**

## 五、2019→2022 独有的新盲区：LUNA / FTT 揭示了什么？

### 1) LUNA 的完整轨迹
- `2019`：历史观察池外
- `2022`：进入 `top5`
- 后续：归零级崩塌

如果只看 2019→2022，LUNA 像极了最标准的新生态超级赢家。

这暴露出一个系统问题：
> **“涨得最猛” 和 “结构最稳” 不是同一件事。**

LUNA 说明，现有系统需要一个新的 Hard fail 维度：
- 不是看它有没有产品
- 而是看它的需求是不是过度依赖**自我增强、闭环维持、脆弱信用结构**

---

### 2) FTT 的完整轨迹
- 2019 资产身份其实不连续（符号重名）
- 2022 作为 FTX Token 冲进前列
- 后续随平台信用一起崩塌

FTT 说明：
> **平台资源驱动本身可以是强规律，但如果资源护城河依赖的是不透明资产负债表或内部信用支撑，它也可能同时是 Hard fail 种子。**

---

### 3) 可以提炼出的新 Hard fail 模式
建议新增一个候选 Hard fail：

> **反身性闭环飞轮 Hard fail**

触发直觉：
- 代币需求高度依赖协议自身飞轮，而不是独立外生需求
- 收益、抵押、流动性、叙事彼此互相支撑
- 一旦飞轮失速，需求会同时蒸发

这和现有 Hard fail 列表里的：
- 纯一次性事件
- 供给极差
- 完全没使用

都不完全一样。它是一个**更危险、也更会伪装成超级赢家**的类型。

---

## 六、DeFi Summer 的赢家衰退，和 4B 是不是同一件事？

我的判断：
> **是同一类时间结构，只是不是同一类资产。**

`4B` 现在的表述更偏：
- ZK / L2 / 技术叙事先行资产

但 2019→2022 的 DeFi Summer 说明，
更广义的模式其实是：

> **创新先行型资产，在第一次大爆发后的 12-18 个月里，会快速从“全员看起来都像赢家”进入“只剩少数入口型/基础设施型仍能维持”的分层阶段。**

所以：
- `COMP / SUSHI / YFI` 的衰退
- `AAVE / UNI` 的相对留存
- `CAKE` 的后段转弱

本质上和 `4B` 是一个母现象的不同资产表达。

这给现有系统的启发是：
> **4B 可以继续保留原表述，但上面最好加一个更广义的母规律：创新浪潮资产普遍存在“爆发 → 分层 → 回落”的时间结构。**

## 七、SOL vs SUI：两个成功新链样本的异同

### 相同点
1. 都是**历史观察池外 / 极早期**进入视野的新链
2. 都不是靠单一技术卖点，而是靠：
   - 链身份清楚
   - 生态扩张明显
   - 用户 / 应用 / 流动性同步增强
3. 都说明：
> **新链成功不是“技术先进”四个字，而是生态飞轮是否真的转起来。**

### 不同点
1. **SOL 所处的是创新牛市**
   - 市场愿意给新链极高估值弹性
   - tokenomics 和供给问题更容易被忽略

2. **SUI 所处的是更筛选、更怀疑的市场**
   - 供给与资本结构更早被放到台前审视
   - 所以它即使成功，也更容易被打上“重融资、要打折”的标签

3. **SOL 的比较对象是旧公链时代的 incumbents**
   **SUI 的比较对象是已经被教育过一轮的新链市场**

结论：
> **SOL 与 SUI 证明的是同一条核心规律（新链生态爆发路径有效），但它们处在完全不同的市场制度环境里：SOL 的胜利更像“抢新大陆”，SUI 的胜利更像“在筛选更严的市场里杀出重围”。**

## 八、对现有系统的最终建议

### 可以升级为“结构性规律”的
1. **高位续涨难 / top50 惯性强**
2. **赛道命运先于个体选择**
3. **入口 / 基础设施位置比泛叙事更能跨周期成立**
4. **创新浪潮会先整体爆发，再快速分层淘汰**

### 保持为“周期特异”的
1. 新公链主导型爆发
2. 平台/分发资源驱动
3. 消费者注意力浪潮
4. RWA / 黄金锚定 / 真实收益偏好

### 现有体系最值得新增的
1. **反身性闭环飞轮 Hard fail**
2. **供给规则的周期敏感性说明**：在 mania 里它更像事后上限约束，在成熟筛选市里它更像前置过滤器

---

## 九、一句话结论

> **2019→2022 与 2023→2026 交叉后，真正最硬的不是“某个赛道永远最强”，而是几条更底层的结构：高位难续涨、赛道先于个体、入口/基础设施更可迁移、创新浪潮一定分层。与此同时，2019→2022 还额外暴露出一个现有体系尚未完整覆盖的盲区：反身性闭环飞轮既能制造超级赢家，也能孕育最危险的 Hard fail。**
