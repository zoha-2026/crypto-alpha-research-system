# Step 2 + Step 3：2019→2022 独立归因与规律提炼 v1

> 前提约束：
> - 这份文档只基于 `2019-03-12 -> 2022-03-12` 的 Step 1 优先研究池做归因与提规律
> - **先独立提炼，不对照现有 2023→2026 的 17 条规律**
> - 归因阶段**不使用 2022 年之后的信息**去回头解释 2019→2022 的上升

主样本来源：`data/step1_historical_2019_2022_v1.csv`

## 一、样本结构速览

- 优先研究池总数：`41`
- 路径 A（有历史排名爬升）：`16`
- 路径 B（强新上位）：`25`
- 高位爬升优先池：`0`

这 3 个结构特征本身已经很有信息量：
1. **强新上位占比极高（25/41）**，说明 2019→2022 是典型的“新物种爆发周期”
2. **高位爬升优先池为 0**，说明 2019 的 top50 到 2022 几乎没有继续显著爬升的旧龙头
3. 路径 A 里大量样本来自 2019 的 `151~500` 深后排，说明这轮历史赢家里混有不少“路径上升成立、但方法上未必可迁移”的老山寨重估样本

## 二、Step 2：按 7 个维度做归因（独立版）

为了避免被现有 17 条规律牵着走，这里不先按旧规则编号整理，
而是先看：**如果只面对 2019→2022 这一池样本，它们自然会分成哪些簇。**

### 新公链 / 生态爆发组

**成员（14）**：FTM、LUNA、SOL、AVAX、DOT、MATIC、ATOM、NEAR、ALGO、HBAR、ICP、EGLD、KLAY、STX

| 维度 | 本组结论 |
|---|---|
| 赛道 / 叙事类型 | L1 / L2 |
| 上涨驱动类型 | 生态扩张驱动 / 混合驱动 |
| 持续性判断 | 长期结构性强势（少数）+ 中期主题强势（多数） |
| 可复制性判断 | 中等可复制 |
| 是否属于可迁移模板 | 是可迁移模板 |
| 供给侧 / Tokenomics | 整体并不普遍干净；这个周期里生态速度和流动性放大常常压过供给约束 |
| 相对基准（BTC + 同赛道） | 大多阶段性明显跑赢 BTC，也显著强于旧公链中位数 |

一句话：链本身成为流动性、开发者和叙事的容器，赢家不是单点功能，而是生态飞轮。

### DeFi / 交易基础设施组

**成员（7）**：UNI、AAVE、CAKE、GNO、SNX、RUNE、REN

| 维度 | 本组结论 |
|---|---|
| 赛道 / 叙事类型 | DeFi / 交易基础设施 |
| 上涨驱动类型 | 产品驱动 / 用户增长驱动 / 混合驱动 |
| 持续性判断 | 长期结构性强势（头部）+ 中期主题强势（其余） |
| 可复制性判断 | 高可复制到中等可复制 |
| 是否属于可迁移模板 | 是可迁移模板 |
| 供给侧 / Tokenomics | 基础设施型头部更能承压；高排放和纯激励型协议的持续性更弱 |
| 相对基准（BTC + 同赛道） | 头部协议通常能跑赢 BTC 与 DeFi 中位数，弱者到 2022 已明显掉队 |

一句话：真正能留下来的不是“谁先火”，而是谁在用户入口、费用流或关键交易环节里站住了位置。

### 平台 / 分发资源组

**成员（3）**：LEO、FTT、CHSB

| 维度 | 本组结论 |
|---|---|
| 赛道 / 叙事类型 | 平台币 / 其他 |
| 上涨驱动类型 | 平台资源驱动 / 流动性-上所驱动 |
| 持续性判断 | 中期主题强势到长期结构性强势 |
| 可复制性判断 | 低可复制 |
| 是否属于可迁移模板 | 部分可迁移，需要附加条件 |
| 供给侧 / Tokenomics | 销毁/回购、库存管理、内部资源分配都很关键，但外部难完全穿透 |
| 相对基准（BTC + 同赛道） | 一旦平台业务处于扩张期，往往强于同赛道大多数资产 |

一句话：在创新牛市里，分发本身就是护城河；谁掌握流量、流动性和用户入口，谁就能给代币更高弹性。

### 消费者注意力 / 游戏 / Meme 波组

**成员（6）**：MANA、SHIB、SAND、AXS、FLOW、BTT

| 维度 | 本组结论 |
|---|---|
| 赛道 / 叙事类型 | Meme / 事件币 / 其他 |
| 上涨驱动类型 | 用户增长驱动 / 情绪-事件驱动 / 混合驱动 |
| 持续性判断 | 中期主题强势 + 短期事件强势 |
| 可复制性判断 | 低可复制 |
| 是否属于可迁移模板 | 不是稳定可迁移模板 |
| 供给侧 / Tokenomics | 供给往往不是第一约束，注意力与社区扩张会暂时掩盖它 |
| 相对基准（BTC + 同赛道） | 主题爆发时显著跑赢 BTC，但一旦热度降温就更容易失去相对强度 |

一句话：这个周期证明，消费者叙事足够强时，文化和注意力本身就能把资产推进 top50。

| 相对基准（BTC + 同赛道） | 先跑赢小赛道中位数，才能借牛市 beta 进入更大共识区 |

一句话：不属于最热主线也能赢，但前提是它真占了一个“别人替不了”的基础设施位。

一句话：这组提醒我们：历史强势币里总会混入一部分“路径上赢了、但方法上学不到太多”的样本。

## 三、Step 3：只看 2019→2022，会独立得出的 8 条规律

### 规律 A：2019→2022 最强主线不是旧龙头慢爬，而是**新公链 / 新生态整体爆发**
证据：LUNA、SOL、AVAX、DOT、MATIC、ATOM、NEAR、ALGO、HBAR、EGLD、KLAY、STX，几乎把这一轮最典型的新生态扩张样本都包进来了。

独立结论：
> **当一个周期的主驱动力是“新生态扩张”时，排名跃升最强的往往不是已有头部资产，而是能承接开发者、流动性和用户迁移的新链 / 新网络。**

---

### 规律 B：这个周期的真正大赢家，很多来自**历史观察池外的新物种**
证据：`25 / 41` 的优先样本走的是路径 B，而且大多直接冲进 `top50`。

独立结论：
> **2019→2022 的 alpha 主要来自“新东西”，不是“老东西变更强一点”。**

---

### 规律 C：平台 / 分发 / 流动性入口，本身就是强势路径
证据：LEO、FTT、CHSB 都不是最典型的协议创新币，但都在周期里拿到了极高弹性。

独立结论：
> **在创新牛市里，谁控制流量、分发、上所和用户入口，谁就能把代币做出远高于基本面线性增长的弹性。**

---

### 规律 D：消费者注意力可以制造 top50 级强势，但这类胜利的可复制性明显弱于生态/基础设施胜利
证据：MANA、SHIB、SAND、AXS、FLOW、BTT 全部来自“人会涌进去”的赛道，而不是最硬的底层产品逻辑。

独立结论：
> **2019→2022 里，注意力和文化本身就是增长引擎；但它更像主题浪潮，不像稳定模板。**

---

### 规律 E：DeFi 的胜负，不在“有没有 DeFi 叙事”，而在“是不是核心入口 / 核心基础设施”
证据：UNI、AAVE、GNO、RUNE、SNX 还能留在优先池；而这一轮真正更难穿越的，往往是纯流动性激励、治理壳、或农场复制品。

独立结论：
> **DeFi 不是一个整体赢家模板；真正能跑到 2022 还站得住的，是头部入口、关键基础设施和拥有网络位置的协议。**

---

### 规律 F：高位 incumbents 的惯性极强，哪怕是在大牛市，旧 top50 也很难继续进入优先研究池
证据：2019→2022 的高位爬升优先池是 `0`。连 BNB、ADA、LINK、CRO、DOGE 这种旧 top50 里确实又涨过的币，也只够进入**全量增长池**，进不了**优先研究池**。

独立结论：
> **高位继续显著上升，是罕见事件，不是默认路径。**

---

### 规律 G：2019→2022 这个周期里，供给不是第一筛选器；生态速度、流动性和叙事强度可以暂时压过 tokenomics
证据：LUNA、SHIB、ICP、CAKE、AXS 这类样本说明，至少在这个周期里，“供给不完美”并不妨碍资产冲进前排。

独立结论：
> **在创新扩张周期，tokenomics 更像事后脆弱性来源，而不是最早阻止上升的门槛。**

---

### 规律 H：闭环飞轮 / 反身性系统，会给出这个周期里最陡峭的上升曲线
证据：LUNA、FTT，以及部分依赖平台飞轮或高激励循环的样本，都体现出“价格上涨、流动性增加、叙事强化、用户再进入”的自我增强结构。

独立结论：
> **2019→2022 最陡峭的赢家，很多不是靠慢慢证明基本面，而是靠一个足够强的闭环飞轮把排名迅速拉升。**

注意：这条在 2022 当下只能说明“上升机制很强”，**不能仅凭 2022 之前的信息直接把它定义成失败模式**。

## 四、如果只看 2019→2022，我会先记住什么？

### 最硬的 4 个独立印象
1. **新生态爆发 > 旧龙头慢爬**
2. **路径 B（新上位）是主战场**
3. **DeFi 要分层：入口型/基础设施型强于一切都算 DeFi**
4. **平台/分发能力和闭环飞轮，在这个周期里是真正的大杠杆**

### 最该保持克制的 3 个地方
1. 不要把消费者注意力浪潮误判成稳定模板
2. 不要把所有低位爬升老币都当成可迁移经验
3. 不要以为 2019→2022 的经验能直接证明“供给不重要”——它更可能只是说明**这个周期允许供给问题被暂时掩盖**

## 五、额外观察（为后续交叉比对做准备，但这里先不对照现有规则）

### 1) LUNA 与 FTT 是“上升机制很强”的极端样本
在 2022 当下看，它们最突出的特征不是“风险”，而是：
- 飞轮强
- 分发强
- 市场承接强
- 叙事强化快

这意味着后续交叉比对时，应该特别问：
> **现有体系有没有足够描述“飞轮很强，但脆弱性也很高”的规则？**

### 2) DeFi Summer 的赢家，到 2022 已经明显分层
这说明我们后续做交叉比对时，应该问：
> **“创新先行 → 分层淘汰 → 只剩基础设施和入口”**，是不是一个比“技术叙事回落”更广义的周期模式？

### 3) SOL 是这个周期最标准的生态爆发样本
它的意义不仅是涨得快，
而是它把“链本身作为流动性容器”这件事体现得非常完整。

后面做跨周期比较时，最值得拿它去对照的，就是另一个真正成功的新链样本。
